# План доработки frontend MovieSearch

## Дата создания плана
2026-06-23

## Цели доработки
Улучшить внешний вид и функциональность frontend без изменения FastAPI backend.

---

## Требования

### 1. Улучшить баннер (hero-секцию) на главной странице
**Текущее состояние**: Однотонный градиентный баннер
**Желаемое состояние**: Красочный баннер с анимациями и эффектами

#### План изменений:
- Добавить динамические анимированные градиенты и круги
- Реализовать эффект плавного движения фона
- Улучшить typography заголовка с градиентом и анимацией
- Добавить больше визуальной глубины через тени и слои

#### Файл для изменения:
- `frontend/src/pages/Home.vue`

---

### 2. Сделать рекомендации размытыми для неавторизованных пользователей
**Текущее состояние**: Все пользователи видят рекомендации в полном объеме
**Желаемое состояние**: Неавторизованные пользователи видят размытый контент

#### План изменений:
- Добавить проверку авторизации в компоненте Recommend.vue
- Применить CSS-эффект `filter: blur()` для неавторизованных
- Добавить призыв к авторизации поверх размытого контента
- Сохранить возможность просмотра формы запроса

#### Файл для изменения:
- `frontend/src/pages/Recommend.vue`

---

### 3. Добавить возможность добавления в избранное с главной страницы
**Текущее состояние**: Кнопка добавления в избранное работает только в рекомендациях
**Желаемое состояние**: Кнопка доступна везде, включая главную страницу

#### План изменений:
- Убедиться, что MovieCard.vue корректно работает на главной странице
- Проверить логику проверки авторизации и добавления в избранное
- Убедиться, что демонстрационные фильмы имеют соответствующий статус

#### Файлы для проверки:
- `frontend/src/pages/Home.vue`
- `frontend/src/components/MovieCard.vue`

---

### 4. Улучшить общий дизайн
**Текущее состояние**: Стандартный современный дизайн
**Желаемое состояние**: Более "крутой" и современный дизайн

#### План изменений:
- Улучшить цветовую палитру и градиенты
- Добавить более выраженные анимации и переходы
- Улучшить карточки фильмов с лучшими эффектами при наведении
- Добавить современные эффекты (glassmorphism, micro-interactions)

#### Файлы для изменения:
- `frontend/src/pages/Home.vue`
- `frontend/src/pages/Recommend.vue`
- `frontend/src/components/MovieCard.vue`
- `frontend/src/style.css` (опционально)

---

### 5. Обеспечить корректную загрузку обложек фильмов
**Текущее состояние**: Используется дефолтная обложка с Yandex
**Желаемое состояние**: Фильмы загружаются с обложкой, которая приходит с бэкенда

#### План изменений:
- Убедиться, что `poster_image` из API отображается корректно
- Оставить дефолтную обложку как fallback при ошибке загрузки
- Проверить, что `MovieCard.vue` использует правильное поле

#### Файлы для проверки:
- `frontend/src/components/MovieCard.vue`
- `frontend/src/types/index.ts`

---
