# Movie Advisor API

A REST API for AI-powered movie recommendations built with FastAPI and Gemini.

The service accepts a user's preferences, mood, genres, or a free-form description and returns personalized movie recommendations generated by Google's Gemini model. The project is being developed as a learning exercise focused on backend development, API design, AI integration, and working with external APIs.

---

## Features

- AI-powered movie recommendations using Gemini
- Free-text prompt support
- Request validation with Pydantic
- Modular project structure
- Async request handling
- Interactive API documentation
- JWT authentication (in development)
- Database support with SQLAlchemy and Alembic

## API Endpoints

### Get Movie Recommendations

**POST** `/movies/recommend`

#### Request

```json
{
  "prompt": "I want a sci-fi movie with time travel and a mind-blowing plot"
}
```

#### Response

```json
{
  "prompt": "I want a sci-fi movie with time travel and a mind-blowing plot",
  "movies": [
    {
      "title": "Interstellar",
      "genre": "Science Fiction",
      "year": 2014,
      "description": "A team of explorers travels through a wormhole in search of humanity's future.",
      "poster": "https://example.com/interstellar.jpg"
    },
    {
      "title": "Tenet",
      "genre": "Science Fiction",
      "year": 2020,
      "description": "An agent manipulates time to prevent global catastrophe.",
      "poster": "https://example.com/tenet.jpg"
    }
  ]
}
```

## Installation

### Clone the repository

```bash
git clone https://github.com/Vladislav-Holod/book_search.git
cd book_search
```

### Create a virtual environment

```bash
python -m venv .venv
```

### Activate the environment

Linux / macOS:

```bash
source .venv/bin/activate
```

Windows:

```powershell
.venv\Scripts\activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Configure environment variables

Create a `.env` file:

```env
DATABASE_URL=postgresql+asyncpg://user:password@localhost/movie_db

SECRET_KEY=your_secret_key
ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=30

GEMINI_API_KEY=your_gemini_api_key
```

### Run the application

```bash
uvicorn app.main:app --reload
```
